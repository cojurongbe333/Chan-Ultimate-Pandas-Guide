## 🧪 Getting Started

1. **Clone this repo**  
```bash
git clone https://github.com/yourusername/pandas-mastery.git
cd pandas-mastery
```

2. **Install the dependencies**  
```bash
pip install -r requirements.txt
```

3. **Launch and explore!**  
```bash
jupyter lab
```
